## Usage

The accepted input format is a text file

Run mars-rover pathToFile.txt

```bash
mars-rover pathToFile.txt
```

## File Format example
```txt
Plateau:5 5
Rover1 Landing:1 2 N
Rover1 Instructions:LMLMLMLMM
Rover2 Landing:3 3 E
Rover2 Instructions:MMRMMRMRRM
```
